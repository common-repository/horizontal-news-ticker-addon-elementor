Horizontal News Ticker Addon - Elementor is a flexible and easy to use news ticker plugin for WordPress. This plugin will help you to make horizontal news ticker in your website. You can show ticker by post category or by post tags or by creating custom texts with custom links. You can write custom text in WP default post editor.

== DESCRIPTIONS ==
Horizontal News Ticker Addon is a plugin to gives users a extra-options to add horizontal ticker section anywhere by Elementor the login page. You can control the speed of scrolling and typography. You can display ticker in your posts/pages by shortcode. You can even use the shortcode in your theme file by inserting it wherever you want to show ticker.
